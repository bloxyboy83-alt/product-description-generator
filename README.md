# 🛍️ Product Description Generator

A lightweight, customizable tool for generating high-quality, aesthetic product descriptions for online shops, merch, and creative storefronts.

Designed for creators, small shop owners, and digital artists who want fast, consistent, and engaging product copy without overthinking every line.

---

## ✨ What This Does

This tool takes simple inputs like:

- Product name  
- Vibe / aesthetic style  
- Key features  

…and turns them into polished, marketing-ready product descriptions.

Perfect for:
- 🛍️ Online merch shops  
- 🎨 OC / character-themed stores  
- ✨ Aesthetic product branding  
- 📦 Digital or physical goods listings  

---

## 💡 Example

**Input:**
```text
Name: Kaitie sniffing a flower  
Vibe: Very cute  
Features: double-sided print, glitter finish, durable acrylic

Output:

✨ Kaitie sniffing a flower ✨

A very cute piece designed to stand out and feel special.

Features include: double-sided print, glitter finish, durable acrylic.

Perfect for collectors, aesthetic lovers, and anyone who wants something that feels a little magical in their everyday life.

⚙️ Installation

Clone the repository:

git clone https://github.com/yourusername/product-description-generator.git
cd product-description-generator
🚀 Usage

Run the script:

python main.py

Or import it into your own project:

from main import generate_description

print(generate_description(
    name="Your Product Name",
    vibe="cute / dreamy / dark / luxury",
    features="feature 1, feature 2, feature 3"
))
🧠 How It Works

This project uses a simple template-based generator.

You provide:

A product name
A vibe (tone of voice / aesthetic style)
A list of features

The generator combines them into structured, readable, and aesthetic product copy designed for online storefronts.
